# Installation

## Requirements

- Python 3.10 or higher
- pip

## Install from PyPI (recommended)

```bash
pip install repo2readme
```

## Install from source

If you want the latest changes from `main`, or plan to contribute:

```bash
git clone https://github.com/agsaru/repo2readme.git
cd repo2readme
pip install -e .
```

## Verify the install

```bash
repo2readme --help
```

You should see the list of available commands (`run`, `reset`) and their options.
